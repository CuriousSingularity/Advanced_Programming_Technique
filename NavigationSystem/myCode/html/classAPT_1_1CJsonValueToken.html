Detailed Description

template<CJsonToken::TokenType tokenType, class T>
class APT::CJsonValueToken< tokenType, T >

This template is used to define special tokens that have an associated value of the given type. This is not intended to be used by programmers. Use the derived types CJsonStringToken, CJsonNumberToken and CJsonBoolToken instead.
