CPoiDatabase &  poiDb,
MergeMode  mode

Fill the databases with the data from persistent storage. If merge mode is MERGE, the content in the persistent storage will be merged with any content already existing in the data bases. If merge mode is REPLACE, already existing content will be removed before inserting the content from the persistent storage.
